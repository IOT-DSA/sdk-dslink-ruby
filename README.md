# DSLink SDK for Ruby 

DSLink SDK for Ruby

## Getting Started

### Install

Add the following to your Gemfile:

```bash
gem 'dslink', :git => 'git://github.com/IOT-DSA/sdk-dslink-ruby.git'
```


### Prerequisites

- [Git](https://git-scm.com/downloads)
- [Ruby](https://www.ruby-lang.org/en/downloads/)
